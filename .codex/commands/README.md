# Nightgauge Codex Slash Commands

These files define custom Codex CLI slash commands.

Install locations:

- Project-local: `.codex/commands/`
- User-global: `~/.codex/commands/`

Supported commands:

- `/nightgauge-issue-pickup`
- `/nightgauge-feature-planning`
- `/nightgauge-feature-dev`
- `/nightgauge-feature-validate`
- `/nightgauge-pr-create`
- `/nightgauge-pr-merge`
