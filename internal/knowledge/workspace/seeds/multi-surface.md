# Product Surfaces

Use this page to document the user-facing surfaces of the repository being
developed.

| Surface              | Users | Responsibilities | Status |
| -------------------- | ----- | ---------------- | ------ |
| Command line         |       |                  |        |
| Editor or desktop UI |       |                  |        |
| API or SDK           |       |                  |        |

## Shared behavior

- Which domain rules must remain consistent across surfaces?
- Where is the authoritative implementation?
- What public contract connects the surfaces?
- How does each surface behave offline or when an optional dependency fails?

Do not record confidential customer, commercial, or private-service details in
a repository knowledge base.
