# cross-repo/

Workspace-level knowledge for contracts and conventions shared by two or more
repositories.

Good entries include:

- Public API and event contracts
- Shared package ownership and compatibility rules
- Authentication boundaries
- Cross-repository version matrices

Do not place credentials, customer data, private deployment details, commercial
plans, or confidential roadmaps here. Keep entries terse and point to the
authoritative public source rather than duplicating it.

`[[cross-repo:platform-api-contract]]` resolves to
`.nightgauge/knowledge/cross-repo/platform-api-contract.md`.
