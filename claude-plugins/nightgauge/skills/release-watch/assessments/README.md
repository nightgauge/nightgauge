# Feature Assessments

Release Watch can create repository-specific assessments in this directory.
Use [TEMPLATE.md](TEMPLATE.md) to evaluate an upstream feature against the
repository's current architecture, safety requirements, and maintenance cost.

Assessments are planning artifacts and are private by default. Before
publishing one, remove customer data, private repository details, commercial
information, unreleased roadmap commitments, and internal prioritization.

Nightgauge's historical company assessments are maintained outside the public
repository.
