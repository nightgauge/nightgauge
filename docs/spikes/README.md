# Spike Artifacts

Nightgauge can materialize repository-specific research artifacts in this
directory. A spike records a bounded question, evidence, recommendation, and
machine-readable recommendation block as described in
[SPIKE_CONTRACT.md](../SPIKE_CONTRACT.md).

This public repository does not publish the Nightgauge company's historical
research corpus. New artifacts must be reviewed for credentials, customer data,
commercial information, private infrastructure, and unreleased roadmap details
before they are committed.
